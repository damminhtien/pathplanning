# Security Policy

## Supported Versions
We support the latest main branch and the most recent PyPI release.

## Reporting a Vulnerability
- Email the maintainers or open a private security advisory on GitHub.
- Provide a minimal reproduction and impact assessment if possible.
- We will acknowledge receipt within 5 business days and coordinate a fix or workaround.

## Expectations
- Please avoid public disclosure until a fix is available.
- We will credit reporters unless they request anonymity.
